# Library Management — TypeScript Reference Implementation

- Node 20+
- TypeScript (strict, `noUncheckedIndexedAccess`)
- Vitest for the test runner

## Build & Run

```bash
cd library-management/typescript
npm install
npm test
```

See [`WALKTHROUGH.md`](WALKTHROUGH.md) for the design rationale and [`../SCENARIOS.md`](../SCENARIOS.md) for the shared specification.
