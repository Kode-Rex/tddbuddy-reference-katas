# CSV Query — Python Reference Implementation

- Python 3.11+
- pytest for the test runner

## Build & Run

```bash
cd csv-query/python
python3 -m venv .venv
.venv/bin/pip install -e '.[dev]'
.venv/bin/pytest
```
