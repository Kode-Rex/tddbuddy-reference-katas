# Bank OCR — TypeScript Reference Implementation

- Node 20+
- Vitest 1.6
- TypeScript 5 strict (with `noUncheckedIndexedAccess`)

## Build & Run

```bash
cd bank-ocr/typescript
npm install
npm test
```

See [`WALKTHROUGH.md`](WALKTHROUGH.md) for the TypeScript-specific notes and [`../SCENARIOS.md`](../SCENARIOS.md) for the shared specification.
