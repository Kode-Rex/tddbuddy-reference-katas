# Jelly vs Tower — TypeScript Reference Implementation

- Node 20+
- TypeScript 5+
- Vitest test runner
- Strict typing (`strict: true`, no `any`)

## Build & Run

```bash
cd jelly-vs-tower/typescript
npm install
npm test
```
