# Kata Potter — TypeScript Reference Implementation

- Node 20+
- TypeScript 5 strict
- Vitest 1.6 for the test runner

## Build & Run

```bash
cd kata-potter/typescript
npm install
npx vitest run
```

See [`WALKTHROUGH.md`](WALKTHROUGH.md) for the TypeScript-specific notes, the [C# walkthrough](../csharp/WALKTHROUGH.md) for the full design rationale, and [`../SCENARIOS.md`](../SCENARIOS.md) for the shared specification.
