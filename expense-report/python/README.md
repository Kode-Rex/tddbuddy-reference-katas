# Expense Report — Python Reference Implementation

- Python 3.11+
- pytest
- Dataclasses for domain types

## Build & Run

```bash
cd expense-report/python
python3 -m venv .venv
.venv/bin/pip install -e '.[dev]'
.venv/bin/pytest
```
