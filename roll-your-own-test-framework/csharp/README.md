# Roll Your Own Test Framework — C# Reference Implementation

- .NET 8+
- xUnit for the test runner
- FluentAssertions for readable assertions

## Build & Run

```bash
cd roll-your-own-test-framework/csharp
dotnet restore
dotnet test
```

See [`WALKTHROUGH.md`](WALKTHROUGH.md) for design rationale and [`../SCENARIOS.md`](../SCENARIOS.md) for the shared specification.
