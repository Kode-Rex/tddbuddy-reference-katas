# Multi-Threaded Santa — TypeScript Reference Implementation

- Node 20+
- TypeScript 5+
- Vitest test runner
- Strict typing (`strict: true`, no `any`)
- Async pipeline with Promise-based bounded queues

## Build & Run

```bash
cd multi-threaded-santa/typescript
npm install
npm test
```

See [`WALKTHROUGH.md`](WALKTHROUGH.md) and [`../SCENARIOS.md`](../SCENARIOS.md).
