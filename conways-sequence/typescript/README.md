# Conway's Sequence — TypeScript Reference Implementation

- Node 20+
- TypeScript 5 strict mode
- Vitest 1.6 for the test runner

## Build & Run

```bash
cd conways-sequence/typescript
npm install
npx vitest run
```

See [`WALKTHROUGH.md`](WALKTHROUGH.md) for the design note and [`../SCENARIOS.md`](../SCENARIOS.md) for the shared specification.
