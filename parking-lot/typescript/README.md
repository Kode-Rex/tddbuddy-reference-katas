# Parking Lot — TypeScript Reference Implementation

- Node 20+
- TypeScript 5 (strict)
- Vitest 1.6

## Build & Run

```bash
cd parking-lot/typescript
npm install
npx vitest run
```

See [`WALKTHROUGH.md`](WALKTHROUGH.md) for the design rationale and [`../SCENARIOS.md`](../SCENARIOS.md) for the shared specification.
