# Roll Your Own Mock Framework — TypeScript Reference Implementation

- Node 20+
- Vitest for the test runner
- TypeScript 5+ strict mode

## Build & Run

```bash
cd roll-your-own-mock-framework/typescript
npm install
npx vitest run
```

See [`WALKTHROUGH.md`](WALKTHROUGH.md) for design rationale and [`../SCENARIOS.md`](../SCENARIOS.md) for the shared specification.
