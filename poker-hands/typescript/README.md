# Poker Hands — TypeScript Reference Implementation

- Node 20+
- TypeScript 5, strict mode (`noUncheckedIndexedAccess`)
- Vitest 1.6 for the test runner

## Build & Run

```bash
cd poker-hands/typescript
npm install
npx vitest run
```

See [`WALKTHROUGH.md`](WALKTHROUGH.md) for the design rationale and [`../SCENARIOS.md`](../SCENARIOS.md) for the shared specification.
