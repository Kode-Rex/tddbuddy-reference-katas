# Balanced Brackets — C# Reference Implementation

- .NET 8+
- xUnit for the test runner
- FluentAssertions 6.12.0 for readable assertions

## Build & Run

```bash
cd balanced-brackets/csharp
dotnet restore
dotnet test
```
